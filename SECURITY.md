# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| 0.1.x   | No        |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Use GitHub's private advisory form:  
https://github.com/lxxyaz-cmd/zion/security/advisories/new

Include:
- A description of the vulnerability and its impact
- Steps to reproduce (proof-of-concept code is helpful)
- Which versions are affected

You will receive an acknowledgement within 72 hours. If the vulnerability is confirmed, we will coordinate a fix and disclosure timeline with you before publishing a CVE.
